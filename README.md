# Go by Example

A collection of annotated Go programs demonstrating the core features of the language, inspired by [Go by Example](https://gobyexample.com/).

## Requirements

- [Go](https://golang.org/dl/) 1.26+

## Structure

Each file is a self-contained program covering a specific concept:

| File | Topic |
|------|-------|
| `00-hello-world.go` | Hello World |
| `01-values.go` | Values (strings, integers, floats, booleans) |

## Running

Run any example directly:

```bash
go run 00-hello-world.go
go run 01-values.go
```
